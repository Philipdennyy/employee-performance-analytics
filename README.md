# employee-performance-analytics
Interactive dashboard for analyzing developer performance, team comparison, and salary fairness using data analytics.
## Developer Performance Analytics Dashboard

This project is a data analytics dashboard designed to evaluate software developer performance within an IT team. It provides insights into individual and team performance, salary fairness, and overall productivity.

### Key Features:
- Developer performance scoring system
- Team-based performance comparison
- Salary fairness analysis (underpaid / overpaid detection)
- Interactive visualizations using Plotly
- Leaderboard for ranking developers
- Optional ML-based performance prediction

### Technologies Used:
- Python
- Streamlit
- Pandas, NumPy
- Plotly
- Mockaroo (for synthetic dataset generation)

### Use Case:
This system simulates how companies analyze employee performance and compensation to support decision-making in HR analytics.
